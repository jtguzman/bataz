# module_bindings/

This directory is generated by:

```bash
spacetime generate --lang typescript \
  --out-dir res://network/module_bindings \
  --project-path server/bataz-server/
```

Do NOT edit these files manually — they are overwritten on every `spacetime generate` run.

## Contents (after generation)

- `DbConnection.ts` — connection builder
- `Game.ts`, `Pawn.ts`, `CardHand.ts`, `DeckState.ts`, `PendingAttack.ts`, `EventLog.ts` — table types
- `Reducers.ts` — typed reducer call wrappers

## Required before SpacetimeClient.ts can run

SpacetimeClient.ts imports from `./module_bindings`. Run `spacetime generate` after every
schema change in `server/bataz-server/Lib.cs`.
